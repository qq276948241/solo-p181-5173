## 1. 产品概述

校园二手书交易平台前端应用，为在校学生提供便捷的二手书买卖服务，解决教材循环利用问题。
- 目标用户：高校在校学生
- 核心价值：低成本获取教材，闲置书籍变现，环保节约

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 模拟登录（ID: 10086） | 浏览书籍、筛选、加购物车、收藏、发布 |

### 2.2 功能模块
1. **首页**：瀑布流书籍列表、左侧学院筛选、价格区间筛选
2. **详情页**：书籍详情、成色信息、卖家昵称、加入购物车
3. **个人中心**：我发布的书、我收藏的书

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 左侧筛选栏 | 按学院多选筛选、价格区间滑块筛选 |
| 首页 | 瀑布流列表 | 响应式多列瀑布流展示书籍卡片，筛选切换时 fade 过渡 |
| 详情页 | 书籍信息 | 封面、书名、作者、价格、成色、学院分类 |
| 详情页 | 卖家信息 | 卖家昵称、加入购物车按钮 |
| 个人中心 | Tab 切换 | "我发布的"和"我收藏的"两个 Tab |
| 个人中心 | 书籍列表 | 复用书籍卡片组件展示 |
| 全局 | 导航栏 | Logo、页面导航、购物车数量徽标（实时同步） |
| 全局 | 登录提示 | 未登录点击收藏时弹出模态提示框 |

## 3. 核心流程

用户进入首页 → 通过左侧筛选栏筛选书籍 → 浏览瀑布流书籍列表 → 点击书籍进入详情页 → 查看详情并加入购物车/收藏（未登录则弹提示） → 进入个人中心查看发布和收藏。

## 4. 用户界面设计

### 4.1 设计风格
- 主色：米黄 #F5E6C8
- 强调色：墨绿 #2F4F4F
- 按钮风格：圆角 8px，墨绿填充主按钮，米黄背景 hover 效果
- 字体：标题使用思源宋体，正文使用思源黑体
- 布局风格：卡片式布局，顶部导航 + 左侧筛选 + 主内容区
- 图标风格：线性简约图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页 | 筛选栏 | 米黄背景卡片，墨绿标签按钮，滑块组件 |
| 首页 | 书籍卡片 | 白色卡片，圆角 12px，阴影 hover，墨绿价格标签 |
| 详情页 | 详情区 | 大图展示，米黄背景分区，墨绿标题 |
| 详情页 | 操作区 | 墨绿填充加购按钮，米黄描边收藏按钮 |
| 个人中心 | Tab 栏 | 墨绿下划线激活态，米黄背景 |
| 导航栏 | 全局导航 | 米黄背景，墨绿文字，购物车徽标红色圆点 |

### 4.3 响应式
- Desktop-first 设计，1280px 以上 4 列瀑布流
- 1024px 3 列，768px 2 列，移动端 1 列
- 移动端筛选栏折叠为抽屉
